# スキルの紹介

## **画像処理**

画像処理 x AIで、[サッカーの解析](https://fusic.co.jp/works/35)などを行っています。実用として、複数の画像処理の組み合わせや、データセット構築自動化などを重視してます。また、Point Cloudのモデル開発も経験があり、NeRFなど3D空間の構築や認識に興味があります。以下の動画は、iPhoneで撮影した動画から人の検出やチーム判定を行い、鳥瞰図に変換した結果です。
